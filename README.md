# Ежедневник

Приложение-ежедневник, в котором вы можете создавать, удалять и фильтровать свои задачи на день.

## Установка

Необходимо установить зависимости:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## Локальный сервер:

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm dev

# yarn
yarn dev

# bun
bun run dev
```

## Продакшн

Собрать приложение для продакшна:

```bash
# npm
npm run build

# pnpm
pnpm build

# yarn
yarn build

# bun
bun run build
```
